PROJECT OVERVIEW

게임의 즐거움을
웹으로 확장하다

실제 Steam 출시 인디 게임 Sledding Game의
공식 웹사이트를 직접 기획하고 제작했습니다.

게임 특유의 겨울 감성과 유머 코드를
웹 인터랙션으로 번역하는 것이 핵심 과제였습니다.

PERIOD 2 Weeks
TOOLS HTML · CSS · JavaScript · GSAP
SCOPE OF WORK
Web Publishing Interaction Design Visual Design Responsive
BACKGROUND

Why Sledding Game?

출시 5일 만에 100,000장 판매를 달성한
Steam 인디 게임의 매력을 웹에서도 자연스럽게 전달하는
랜딩 페이지를 기획하고 제작했습니다.

02 PROJECT GOALS

01

World-Building

게임의 첫인상 전달

게임의 분위기와 핵심 콘텐츠를 직관적으로 전달하여
플레이 전 기대감을 높이는 랜딩 경험을 설계합니다.

02

Steam Conversion

플레이 유도

게임의 특징과 콘텐츠를 자연스럽게 소개하여
Steam 페이지까지 이어지는 사용자 흐름을 설계합니다.

03

Interactive Experience

인터랙티브 경험

GSAP 애니메이션과 반응형 인터랙션을 활용해
게임의 즐거움을 웹에서도 자연스럽게 전달합니다.
